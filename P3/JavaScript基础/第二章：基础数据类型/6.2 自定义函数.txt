

函数的声明

1.用 function 命令声明     （存在函数声明提升）
funtion 命令      函数名称
function    print ( ) {         {}中的内容为函数体
       ......;
}

调用：   print( );   //  调用格式为：函数名 + （）


2.用函数表达式声明       （不存在函数声明提升）
   变量名  赋值符号 
let    print = function( ) {       {}中为函数体
        ......;
}  ;                //注意后面的分号
赋值等号右边为匿名函数，又称函数表达式。

调用:         print( );
